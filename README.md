## 1. 프로젝트 개요

**PPL-Analytics**는 감정적 판단과 추측에 의존하던 기존의 광고 성과 측정 방식을 **데이터 기반의 정량적 분석 체계로 혁신**하는 서비스입니다. 실제 시청자 행동 데이터와 AI 분석을 통해 광고주가 보다 전략적이고 객관적인 의사결정을 내릴 수 있도록 지원하며, AI 기반 자동 리포트 생성으로 **분석 시간을 80% 단축**시키는 것을 목표로 합니다.

### 헥심가치 (Core Value)
* **데이터 투명성**: 실제 시청자 반응 데이터에 기반한 객관적인 PPL 성과를 분석합니다.
* **효율성 극대화**: AI가 자동으로 리포트를 생성하여 분석 시간을 획기적으로 단축합니다.
* **경쟁 인텔리전스**: 동종 업계 벤치마크와 비교하여 상대적 성과를 평가합니다.
* **시각화 중심**: 직관적인 차트와 인포그래픽으로 즉시 이해 가능한 인사이트를 제공합니다.
* **전략적 방향성**: 단순한 성과 리포팅을 넘어 차세대 마케팅 전략을 제안합니다.

## ✨ 핵심 기능 (Key Features)

<details>
  <summary> ✅ 기능 요구사항 전체 보기 (Functional Requirements) </summary>
  <br/>
  
  ### 🔹 사용자 인증 및 권한 관리
  * **소셜 로그인**: Supabase Auth와 Google OAuth 2.0을 연동하여 간편한 인증을 제공합니다.
  * **세션 관리**: JWT 토큰 기반으로 세션을 관리하며 자동으로 갱신합니다.
  * **권한 분리**: 역할 기반 접근 제어(RBAC)를 통해 관리자, 일반 사용자, 게스트 권한을 분리합니다.

  ### 🔹 데이터 입력 및 파일 관리
  * **스마트 카테고리**: 화장품, 식품, 패션 등 10개 주요 업종별로 제품 정보를 수집합니다.
  * **동적 입력 필드**: 선택된 카테고리에 맞춰 추가 정보를 수집하는 맞춤형 폼을 제공합니다.
  * **멀티 포맷 지원**: PDF, DOCX, XLSX, CSV 등 다양한 형식의 파일을 업로드할 수 있습니다.
  * **보안 저장**: 모든 파일은 AWS S3에 암호화되어 안전하게 저장되며, 버전 관리 및 백업 정책이 적용됩니다.

  ### 🔹 다차원 데이터 수집 및 전처리
  * **시청자 행동 분석**: IPTV/OTT 시청 패턴, 인구통계학적 특성, 디지털 성향 등 다각도로 시청자를 분석합니다.
  * **성과 데이터 통합**: 방송사 데이터, 소셜 미디어 언급량, 검색 트렌드를 종합하여 PPL 성과를 측정합니다.
  * **ROI 정밀 계산**: 투입된 매체비와 제작비 대비 매출 증가 및 브랜드 인지도 상승을 정량화하여 ROI를 계산합니다.
  * **데이터 품질 관리**: 통계적 이상치 탐지, 머신러닝 기반 결측값 처리, 데이터 정규화를 통해 데이터의 신뢰도를 확보합니다.

  ### 🔹 인터랙티브 대시보드 및 시각화
  * **실시간 모니터링**: KPI 대시보드를 통해 실시간 성과 지표를 추적하고, 드릴다운 분석으로 상세 데이터를 탐색합니다.
  * **고급 시각화**: D3.js 기반의 동적 차트, 지역/시간대별 반응도를 보여주는 히트맵, 소셜 키워드 분석을 위한 워드클라우드 등을 제공합니다.

  ### 🔹 보고서 내보내기 및 공유
  * **다중 포맷 지원**: 분석 보고서를 PDF, PowerPoint, Excel 등 다양한 포맷으로 자동 생성하여 제공합니다.
  * **웹 링크 공유**: 실시간으로 업데이트되는 동적 웹 리포트 링크를 통해 손쉽게 결과를 공유할 수 있습니다.

## 🤖 AI 아키텍처 및 리포트 생성 파이프라인

본 서비스는 **이중 AI 모델(Dual AI Model)** 과 **카테고리별 프롬프트 오케스트레이션**을 통해 분석의 속도와 깊이를 모두 확보합니다.

### 🔹 이중 AI 모델 전략

| 모델 | 역할 | 처리 시간 | 특징 |
| :--- | :--- | :--- | :--- |
| **1️⃣ Gemini-2.5-flash-lite** | 원시 데이터 구조화, 초기 인사이트 추출, 프롬프트 전처리 | **30초** | 빠른 속도와 비용 효율성 최적화 |
| **2️⃣ Skywork API** | 다중 에이전트를 활용한 심층 전략 분석 및 최종 보고서 생성 | **10-15분** | 다중 에이전트 검증을 통한 고품질 분석 및 권고사항 도출 |

<br/>

### 🔹 8단계 자동 보고서 생성 파이프라인
AI는 다음과 같은 8단계의 체계적인 파이프라인을 통해 종합적인 분석 보고서를 생성합니다.

1.  **요약 (Executive Summary)**: 핵심 성과 지표(KPI) 및 주요 발견사항을 요약합니다.
2.  **시장 분석 (Market Analysis)**: 업종별 PPL 시장 현황과 최신 트렌드를 분석합니다.
3.  **경쟁사 비교 (Competitive Analysis)**: 동종 업계 벤치마크와 성과를 비교 분석합니다.
4.  **타겟 오디언스 분석 (Target Audience Analysis)**: 시청자 그룹을 세분화하고 반응도를 심층 분석합니다.
5.  **PPL 성과 분석 (PPL Performance Analysis)**: 정량적 데이터를 기반으로 PPL 캠페인의 성과를 평가합니다.
6.  **ROI 및 비용 효율성 (ROI & Cost Effectiveness)**: 총 투입 비용 대비 수익률(ROI)을 분석합니다.
7.  **전략적 권고사항 (Strategic Recommendations)**: 분석 결과를 바탕으로 AI가 데이터 기반 개선 방안을 제시합니다.
8.  **향후 계획 (Future Planning)**: 차기 캠페인 전략과 예산 최적화 방안을 제안합니다.

## 🏗️ 시스템 아키텍처 및 API

### 🔹 RESTful API 설계

| Endpoint | 설명 |
| :--- | :--- |
| **`/auth/*`** | 로그인, JWT 토큰 관리, 사용자 권한 검증 API |
| **`/dashboard/*`** | 시청자 데이터, PPL 성과 등 대시보드 데이터 조회 API |
| **`/analysis/*`** | AI 모델 기반의 비동기 분석 요청 및 결과 조회 API |
| **`/report/*`** | 생성된 보고서의 다운로드 및 공유를 위한 API |

<br/>

### 🔹 외부 시스템 연동
정확한 분석을 위해 다양한 외부 데이터 소스와 연동합니다.
* **방송사 API**: 실시간 시청률 데이터를 연동합니다.
* **소셜미디어 API**: Twitter, Instagram, YouTube의 언급량을 수집합니다.
* **검색 트렌드 API**: Google Trends, 네이버 DataLab의 검색량 데이터를 통합합니다.
* **광고 플랫폼 API**: Facebook, Google Ads의 성과 데이터를 연동하여 종합적인 분석을 수행합니다.

### 🔹 비즈니스 목표

* **의사결정 정확도 향상**: 데이터 기반의 객관적 분석을 통해 PPL 캠페인 성과 예측 정확도를 **80% 이상**으로 높입니다.
* **ROI 개선**: 실시간 성과 추적 및 최적화 제안을 통해 광고주의 ROI를 **30% 이상** 개선하는 것을 목표로 합니다.
